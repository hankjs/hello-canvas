# hello-canvas

#### 介绍

**Web前端开发精品课——HTML5 Canvas开发详解**一书的代码实践


#### 软件架构
根据章节划分文件夹

```
├─src                      # 主要代码
│  │                         # c{number} 是按章节划分的目录
│  ├─c2                      # 直线图形
│  ├─c3                      # 曲线图形
│  ├─c4                      # 线条操作
│  ├─c5                      # 文本操作
│  │ 
│  ├─utils                   # 公共函数
│  ├─css                     # 公共样式
│  ├─css                     # 公共样式
│  │ 
│  ├─template.html           # html模版
│  ├─template.ts             # js模版
│  └─out                     # ts编译输出文件夹
│
└─@types                     # 全局ts接口、类型定义
```


#### 安装教程

1.  npm i

#### 使用说明

1.  npm run watch
  - 编译ts文件

2.  npm start
  - 运行服务,在浏览器访问
